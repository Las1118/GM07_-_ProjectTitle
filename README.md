C# 기초 기능들을 활용한 턴제  RPG게임 구현
program.cs로 메인메뉴창을 띄우고 각 cs들로 메인메뉴의 기능들을 구현
Boss.cs로 보스전 구현. dictionary를 이용한 정해진 패턴의 출력으로 각 스킬에 따른 공격 or 힐링 선택 강요
즉사패턴 도입으로 일정 턴 안에 피해를 우겨넣어 파훼시키는 패턴으로 게임의 긴장감과 난이도 증가
expertZone, BeginnerZone, 랜덤으로 각종 몬스터 출현. 스킬창을 불러와 사용하는 형식으로 턴제 전투 진행. 플레이어가 지속할시 무제한으로 사냥터에서 사냥 가능
skill.cs 레벨업을 통해서 얻은 스킬들을 스킬창 형태로 출력. 메인메뉴 - 스킬창에서는 스킬의 정보 확인 가능. 전투페이즈에서는 스킬 정보 확인 대신 사용하는것으로 전투 가능
moster.cs 몬스터의 스텟 설정 공식 생성
training.cs 레벨 10까지 강제되는 훈련으로 기본적인 RPG의 틀을 따라 반복 훈련으로 레벨 성장 강요
Unit.cs 플레이어의 각종 필요한 기능들을 구현해놓은 파일로 가장 기본적인 이름, 체력, 마나, 골드 그리고 경험치부터 시작해서 레벨업, 레벨업을 통한 스킬 저장 등을 구현
inventory.cs 상점에서 구매하여 저장된 아이템을 확인 및 장착할 수 있는 파일로 아이템 장착시 해당 아이템의 능력치만큼 캐릭터의 능력 상승. 이미 장착한 아이템은 다시 선택할 시 해제 가능
shop.cs 상점창으로 각종 아이템 판매. 처음에는 두가지 기본장비 밖에 없지만 if문으로 적정 레벨 달성시 해금되는 장비들로 구성됨
chapter2credit.cs 보스전을 마무리하면 나타나는 크레딧 창으로 챕터1의 종료를 알리고 2의 시작을 예고한다.
